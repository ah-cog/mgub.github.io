gesso

Tool to build basic package managers based on git and to create command-line tools and execution environments.

gesso is command-line tool that provides a modular framework for rapidly building Python. It can be used to create basic package managers based on git and to create command-line tools and execution environments.

gesso does two things:

  1. Provides a generic package manager based on git repositories.

  2. Provides a basic CLI task execution framework.

This provides a framework for rapidly building a command-line environments using commands stored as modules in git repositories (such as on GitHub).
